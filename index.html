Hva er det? Hvordan funker det?

Hva er HTML?

HTML (HyperText Markup Language) er det som bygger opp nettsider. Det bestemmer hvor tekst, bilder, videoer og lenker skal plasseres. Sammen med CSS for stil og JavaScript for funksjoner, gjør HTML det mulig å lage nettsider akkurat slik du vil ha dem.



Hva er CSS?

CSS (Cascading Style Sheets) er språket som gir nettsider stil og design. Det bestemmer farger, skrifttyper, avstander, oppsett og animasjoner. Ved å bruke CSS sammen med HTML, kan du skape visuelt attraktive og responsive nettsider.



Hva er Javascript?

JavaScript er et programmeringsspråk som gjør nettsider interaktive. Det lar deg legge til dynamiske elementer som knapper som reagerer på klikk, animasjoner, skjemaer som validerer brukerinput, og mye mer. Sammen med HTML for struktur og CSS for design, er JavaScript en av de tre hovedteknologiene som brukes til å bygge moderne nettsider og webapplikasjoner.
